# 探索景點

歡迎來到 **ExploreTags 漫旅書籤**的探索景點中心！

我們精心整理了世界各地和臺灣值得一訪的城市寶藏——博物館、古蹟、圖書館、美術館、觀光工廠、鐵道車站，以及特色美食地圖。
每一個景點，都是一次讓你的旅程更有意義的探索機會。

## 景點分類

- **藝文場館**：博物館、美術館、文化中心，沉浸於歷史與藝術之美
- **古蹟遺址**：百年老建築、史蹟公園，感受時間留下的痕跡
- **圖書館**：地方特色圖書館，知識與建築並存的靜謐所在
- **鐵道車站**：老火車站、輕軌站點，鐵道迷必訪的旅行印記
- **觀光工廠**：走進產業故事，認識在地生產文化
- **特色美食**：巷弄小吃、老字號名店，品嚐城市的獨特滋味

## 如何開始探索？

1. 從左側選單選擇你感興趣的景點類別
2. 找到你想打卡的景點，加入旅行清單
3. 出發前往，完成打卡，收集你的旅行寶石
4. 累積足跡，建立屬於你的城市探索地圖

---

> 每一次出發，不只是旅行，而是一場可以被記錄與收藏的冒險。
